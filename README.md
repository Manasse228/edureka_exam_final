#  Property transfer

### Installation

Have node.js installed, then in the project folder:

```
npm install
```

### Development

To run with live-reloading via beefy:

`npm start`

To build:

`npm run build`

This will generate a `bundle.js` file that is pointed to by the `index.html`.
